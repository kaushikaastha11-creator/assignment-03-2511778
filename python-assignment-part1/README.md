# Assignment — Part 1: Python Basics & Control Flow


## Overview

A command-line Student Grade Tracker that manages student data, computes results, and provides a summary report — built using core Python concepts.

---

## Tasks

### Task 1 — Data Parsing & Profile Cleaning (5 marks)
- Clean raw student data (whitespace, casing, type conversion)
- Validate each student name
- Print formatted profile cards
- Print name in ALL CAPS and lowercase for roll number 103

### Task 2 — Marks Analysis Using Loops & Conditionals (8 marks)
- Print each subject with marks and grade label using a for loop
- Calculate total, average, highest and lowest scoring subject
- Simulate marks-entry system using a while loop with input validation

### Task 3 — Class Performance Summary (7 marks)
- Compute average and pass/fail status for each student
- Print formatted class report table
- Print class topper and class average

### Task 4 — String Manipulation Utility (5 marks)
- Strip, Title Case, count, replace, and split an essay string

---

## How to Run
1. Clone the repository
2. Open part1_grade_tracker.ipynb in Jupyter Notebook or Google Colab
3. Run all cells in order

## Concepts Used
- String methods: .strip(), .title(), .split(), .upper(), .lower()
- Type conversion: int()
- for loops, while loops, zip()
- if/elif/else conditionals
- try/except for error handling
- f-strings for formatting
